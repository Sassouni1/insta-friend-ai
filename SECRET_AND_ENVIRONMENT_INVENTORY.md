# SECRET AND ENVIRONMENT INVENTORY

Scope: exact deployed source of all Edge Functions in `supabase/functions/`
(including `_shared/`) plus the Vite client. `.backup*` files were ignored.
No secret VALUES are included — names, usage, and provenance only.

Legend
- **Required**: function fails / throws on start when missing.
- **Optional**: code has a fallback default or the feature degrades gracefully.
- **Type**: SECRET (credential/signing material) vs CONFIG (non-secret setting).
- **Provenance**:
  - `supabase-auto` → auto-injected by any Supabase project (recreate for free).
  - `lovable-auto` → auto-injected by Lovable platform, not portable.
  - `manual-provider` → must be obtained from the third-party provider and set by hand.
  - `user-generated` → arbitrary random string chosen by the operator.
